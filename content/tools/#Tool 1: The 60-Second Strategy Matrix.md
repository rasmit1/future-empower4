# Tool 1: The 60-Second Strategy Matrix
> Use this scoring grid to instantly diagnose whether a new offer should be integrated or separated.

| Evaluation Pillar | Route A: **Integrate** (Score +1) | Route B: **New Stream** (Score -1) |
| :--- | :--- | :--- |
| **1. Brand Fit** | Deepens what you already do well. Fits the same brand voice. | Targets a new audience. Requires a totally unique brand voice. |
| **2. Operations** | Current tech stacks, payment tools, and delivery pipelines support it. | Needs custom workflows, unique tracking, or isolated pipelines. |
| **3. Buyer Persona** | The exact same ideal client buys this to get faster results. | Introduces friction. Mixing these buyers will confuse both pools. |
| **4. Time to Market** | Needs fast validation using assets you already own. | Built as a standalone asset designed to scale or exit later. |

### 📊 Score Diagnostics:
* **Score +2 to +4**: 🟢 **Integrate**. Do not reinvent the wheel. Put it in your current funnel.
* **Score -2 to -4**: 🔵 **New Stream**. Protect your core. Build a clean partition.
* **Score -1 to +1**: 🟡 **Split Decision**. Run as a temporary, isolated pilot before locking it in.
